1.Create git repo and clone it in local.
2.Create code.py and add code to it.(it will save a csv file to a new "data" folder)
3.Now we do "dvc init" (creates .dvcignore,.dvc)
4.Now do "mkdir s3" (Creates a new S3 directory)
5.Now we do "dvc remote add -d myremote S3"
6.Next "dvc add data/".
  - Now it will ask to do : ("git rm -r --cached 'data'' and "git commit -m "Stop tracking data")
  Because initially we are tracking data/ folder from git so now we remove it for DVC to handle.
8.Again we do "dvc add data/" (creates it data.dvc) then "git add .gitignore data.dvc"
9.Now - "dvc commit" and then "dvc push"
10.Do a git add-commit-push to mark this stage as first version of data
11.Now make changes to mycode.py to append a new row in data , check changes via "dvc status"
12.Again -- "dvc commit" and then "dvc push"
13.Then git add-commit-push (we're saving V2 of our data at this point)
14.check dvc/git status,everything should be upto date.
15.Now repeat step 12-14 for V3 of data.
